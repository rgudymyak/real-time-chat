Задача: Real-time чат
Страница 1
Написать real-time чат, который позволит пользователям в общем канале оправлять и получать
сообщения друг друга.
Функционал
— Переписка должна осуществляться в реальном времени;
— Пользователи могут создать свой канал для общения или подключиться к существующему;
— В каждом канале должны отображаться участники переписки;
— Необходимо реализовать поиск юзеров;
— У приложения должен быть приятный UI;
— Создатель канала должен иметь возможность удалять пользователей из канала.

Для того чтобы воспользоватсья приложением у себя на компьютере нужно проделдать следующие действия

1. Склонировать репозиторий: git clone https://github.com/rgudymyak/real-time-chat.git
2. Перейти в папку клиент из главной папки cd client
3. Установить зависимости для клиента npm install
4. Перейти в папку сервер из главной папки cd server
5. Установить зависимости для сервера npm install
6. Можно запускать проекты из папок сервер и клиент. Для сервера npm run start и npm run dev для клиента

Работа приложения
Открываем несколько окон в браузере. В каждом окне переходим по адресу http://localhost:5173/ - создается подключение. Выбираем в поиске нужного пользователя, создаем комнату, либо заходим в уже созданную комнату и начинаем переписку с другим подключенным пользователем в реальном времени.
Пользователь создавший комнату имеет возможность удаления пользователей, подключившихся после него к комнате.
